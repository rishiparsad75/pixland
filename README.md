# PixLand

AI-powered photo management SaaS platform for event photographers.

## Features
- Role-based authentication (Super Admin, Photographer, User)
- QR-based event access
- AI face recognition photo matching
- Bulk photo upload
- Secure album access
- Event-based photo grouping

## Tech Stack
Frontend: React + Vite + Tailwind
Backend: Node.js + Express
Database: MongoDB Atlas
Cloud: DigitalOcean
AI: Azure Face API
